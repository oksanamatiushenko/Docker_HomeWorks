Задание 2: тест на команды Docker
 1. Создание и запуск нового контейнера из Docker образа:
- docker run
 2. Запуск остановленного контейнера:
- docker start
 3. Принудительное удаление контейнера, даже если он запущен:
- docker rm -f
 4. Отображение списка работающих контейнеров:
- docker ps
 5. Просмотр вывода журналов контейнера:
- docker logs
 6. Отображение списка процессов, работающих внутри контейнера:
- docker top
 7. Предоставление подробной информации о контейнере в формате JSON:
- docker inspect
 8. Присоединение к работающему контейнеру и взаимодействие с ним через стандартные ввод/вывод:
- docker attach
 9. Создание нового Docker образа на основе изменений в контейнере:
- docker commit
 10. Удаление одного или нескольких образов Docker с хостовой системы:
- docker rmi
